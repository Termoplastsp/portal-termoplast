Portal de Sistemas — Configuração inicial

1 Criar projeto no Firebase
1. Acesse console.firebase.google.com
2. Clique em Adicionar projeto
3. Nome: termoplast-portal → Continue → Continue → Criar projeto
4. Aguarde criar e clique em Continuar
2 Ativar o Realtime Database
1. No menu lateral esquerdo, clique em Criação → Realtime Database
2. Clique em Criar banco de dados
3. Localização: escolha Estados Unidos (us-central1)
4. Regras de segurança: selecione "Iniciar no modo de teste" → Ativar
3 Copiar a URL do banco
Na página do Realtime Database você verá uma URL assim:

https://termoplast-portal-default-rtdb.firebaseio.com/

Copie essa URL completa (incluindo o https:// e a / final).
4 Ajustar regras de acesso
1. No Realtime Database, clique na aba Regras
2. Substitua o conteúdo por:

{
 "rules": {
  ".read": true,
  ".write": true
 }
}

3. Clique em Publicar

⚠️ Isso permite leitura/escrita pública. Para produção, proteja com autenticação Firebase.
5 Tudo pronto!
✅ Firebase configurado com sucesso.

Agora todos os dispositivos que abrirem este portal verão os mesmos sistemas em tempo real. Qualquer alteração feita pelo admin aparece instantaneamente para todos.
